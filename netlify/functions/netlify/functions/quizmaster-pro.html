💳 Stripe Ödəniş Sistemi

Quraşdırma təlimatı

1. stripe.com-da hesab açın
2. Dashboard → Developers → API Keys bölməsinə keçin
3. Secret key-i kopyalayın (sk_live_... və ya sk_test_...)
4. Netlify Dashboard → Site settings → Environment variables → STRIPE_SECRET_KEY kimi əlavə edin
5. Saytı yenidən deploy edin

Backend funksiyası vasitəsilə yoxlanılır…

Qeyd: Təhlükəsizlik üçün Stripe açarı bu admin paneldən deyil, birbaşa Netlify-da environment variable kimi saxlanılır. Açar heç vaxt brauzerə göndərilmir.
